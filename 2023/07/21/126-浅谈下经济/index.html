浅谈下经济

发表于 2023-07-22 阅读次数： 本文字数： 1.5k 阅读时长 ≈ 5 分钟

最近一则消息引爆了大多数人的神经：

借着这个机会，我浅谈下我对经济的了解。

  1. 现有经济模式下，资本永不倒。
  2. 资本对利息的认可
  3. 资本的流动趋向固定
  4. 新时代的世袭制

1. 现有经济模式下，资本永不倒。

如果一个人每年需要花100万，那么假设他活了一百岁，那么他需要1亿。
但是如果一个人一出生就有1个亿的话，按照每年1%的利率，他每年可以花100万，死之后还会有1个亿。
从这两个假设我们可以看出——资本永不倒。在现有的经济模式下，更多的是维护资本。如果你有10w，那么你只能买买理财，如果你有1亿，那么你可以跟银行谈利息，谈保本，谈投资。这就是资本的力量，资本的力量不依靠任何人或物，甚至法。现有的道德和社会风气已经在人们心中扎下了这个观念，1个亿的人可以跟官员打交道，可以做一些违法犯法的事情，为啥？因为这个社会发展需要一个引子，而这1亿就是引子。
我之前说过经济的流动，投入1亿，然后转1圈之后，会变成1.2亿，2000万分配到中间环节。随着中国经济的发展，从当初的几亿，到现在的几万亿。资本是不断的膨胀的，为了社会稳定，为了经济稳定，只能维护这些资本，让这些本继续这种钱生钱的游戏。大资本获得的收益肯定是最大的。
所以只要现有经济制度不变的情况下，那么资本永远不倒，至于你说国家收割民营资本，我只能说那是上层分配的事情，跟你我关系不大。只要现在资本制度不变情况下，那么永远都有大资本，大资本永远不倒。

2. 资本对利息的认可

利息在古罗马是不道德的，因为他相当于不劳而获，但是随着资本的洗脑，现在利息是名正言顺，甚至是合情合理的东西，然而正如我上面讲述的，只要1个亿进行放债，那么法律会保护这些钱的安全，利息会让他永远有收益，那么就陷入了一个无风险的稳定收益行为，再加上那几年地方融资开出的15%甚至20%的利率——这就是一个下金蛋的鹅。这次置换利息解决的就是当初的高额利息行为，通过降低利率来变相降低压力，但是对应的风险就是将这些债权收归国有，简单说就是这只鹅现在归银行了。
然而债务是无法靠发行债务解决，核心在于经济发展产生利润。
解决债务的核心依旧是增长，通过高增长率覆盖掉那固定下来的负债利率，利润分配是一回事，无论利润是还债还是给自己，还是要解决每年5%的利息，剩下的才是利润。债务利息就像是悬在他们脖子上的刀子，随时要夺走他们的生命。但是这把刀有着社会法律上的认可，是理所应当的在你脖子上。

3. 资本的流动趋向固定

资本的趋利性，往往会让资本往最赚钱的地方流动。前几年在房地产，这几年在银行。资本流动的固定本质就是行业盈利能力的体现，一个行业大量的进驻资本，那么这个行业肯定有可取之处。而如今信息化的高速发展，导致信息壁垒的消失，尤其是金融投资行业，信息是最快速的。所以这更加了加剧了资本趋向的一致性，那么就会出现进来时候百花齐放，走的时候一地鸡毛。

4. 新时代的世袭制

资本就是新时代的世袭制度，只要新的接手人不要瞎搞，靠着信息和利息增长，那么就会保持住自己的地位。现在的GDP只剩下5%甚至更低，所以不用担心贬值问题。资本的世袭就是在快速发展的阶段，资本获得最大收益，在稳定阶段，资本可以依靠利率和现有产业分布，那么他也可以做到保持自己地位不变的情况。唯一出问题的都是做实体的，开公司的。资本付出的代价是什么，是一定的收益折损，发展时候，实体行业收益20%，愿意付利息12%给资本，稳定时候，实体由于过大的资产无法降低开销，但是资本这时候选择银行，1年的利率足够他们熬过寒冬。

总结

资本可以是任何东西，可以是现金，也可以是随时变成钱的古董，房子。但是我并不是抨击资本，我只是想说明，在如今一套经济运行的逻辑下，只要国家不倒，那么就一直可以这么玩。那么改变如今现状的可能性，就是打破这些现有的经济规则，我们需要一套新的经济规则，自上而下，改变所有人的内心想法。
当然这中间是需要很长的时间，甚至需要一些战争，但是我对经济的补丁一直是不看好。

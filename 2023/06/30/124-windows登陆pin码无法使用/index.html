windows 无法使用pin码登陆

发表于 2023-07-01 阅读次数： 本文字数： 888 阅读时长 ≈ 3 分钟

今天遇见一个十分无语得事情：进入Windows得安全模式，提示：pin码无法使用，需要重新设置，点击重新设置却又没有任何效果。安全模式下照相机驱动都没有，无法使用hello world得脸部登陆，于是就陷入了一个尴尬得界面。
安全模式下甚至无法使用网络，这就意味着，如果你选择重新安装系统保留程序，它会提示你登陆你的微软账号，但是由于没有网络，所以它无法验证你的密码。最后我深思了一下，总结了一下发生得原因和解决方案。

原因

如果你恰好有个微软账号，且买电脑时候没有进入本地账号，或者长期是用微软账号作为自己日常使用得电脑得用户，那么一旦进入安全模式就会出现这个问题。
我得电脑是yoga 14s AMD锐龙版本，最近遇见得问题是重启。于是在微软社区查找了一番之后，找到这么一个问题和我得情况很像。
根据建议，我选择了干净启动：

打开CMD，输入


打开启动控制，选择基本启动，或者诊断启动。

本意是为了排除系统其他程序对我的影响，确定重启的原因。
但是重启之后就陷入了一个死循环，系统没有启用摄像头和网络服务，由于我这台电脑当初直接登陆的是微软账号，所以系统提示我得pin码无法使用，但是我点击设置pin码却没有任何反应，因为大多数网络账号相关服务并没有启动，所以我就很尴尬的卡在那里，无法设置，也无法进入Windows。

解决

首先你需要找到一个U盘，安装一个PE系统。
然后修改BIOS设置，进入PE系统。然后使用Dism ++ 打开一个本地账号，再重启电脑即可。

进入系统之后，按照之前操作，把所有的服务都启动起来，再重启，切换到你日常使用的微软账号即可。后续如果害怕麻烦可以关闭本地账号。
本质是一些网络和账号登陆的服务没有被打开，导致无法进入Windows，所以我先通过本地无密码的账号进入系统，修改掉启动项就解决了。

总结

总结下来就是这个问题好坑，我按照社区的教程进行操作没想到进入另一个循环，而且一开始我还想一个个服务启动，后面发现太多了，于是就直接选择新建本地账号进行操作。
而且最后我还是没有解决网银导致的重启问题，我于是卸载掉所有的网银助手看看后续如何，那个致使系统关机的文件还在那里。
Windows作为一个系统来说，现在越来越复杂，而且对于国内网银的支持真的是一言难尽，一个劲的推崇自己的Edeg浏览器，然而大家还是以ie为主。

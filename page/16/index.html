关于vps的一些设想

发表于 2022-05-02 本文字数： 2.3k 阅读时长 ≈ 8 分钟

首先，描述一下我的服务器配置

  • cpu：1核

  • 内存：512M

  • 硬盘：10G

  • 带宽：1G

  • 流量：1000G/月

这个是搬瓦工的512M的洛杉矶CN2线路，访问国内速度很快，电信联通尤其好，移动差劲。现在这款已经下架，目前最便宜的要1G内存20G硬盘-49.99美元/年，这个只能说当初年少无知，没想到捡了一块宝。

但是相对的也存在问题：我们服务器内核不足，而且内存不足，所以很多大型的服务我们是运行不起来的，尤其不建议运行Mysql服务，虽然通过虚拟化内存可以增加内存容量，但是1核心的cpu始终是系统短板。所以系统搭载以轻快+低负载为主。曾经想过将其变为一个网上影院，后面折腾来折腾去，发现不仅仅是硬盘不足，还有由于cn2网络的不稳定，有时候还不如挂代理直接访问一些视频网站舒服，毕竟人家有优化，有线路。那么综合下来，目前服务器上仅仅搭载3个服务：

  • V2ray：翻墙工具，用来访问一些无法访问的东西。

  • typecho博客：轻量级博客，很适合这种低内存的小主机，主要是不需要安装数据库。

  • clibre-web电子书：在线阅读工具，更新迅速还开源。

其中v2使用脚本安装，安装在系统内部。typecho使用宝塔面板。clibre-web使用python项目管理部署。

现在想重新整理这些服务，毕竟当初安装的时候试错和其他服务，所以导致整个系统的文件有点混乱，包括当初为了独立出一个python，也试过python共存。还有就是宝塔面板更新需要手机绑定等问题，导致一直使用旧的宝塔面板。所以现在想要从头梳理一下自己需求，从系统层面重新搭建一个干净的服务器。

1.系统选择

目前是使用centos系统，当时搬瓦工可以安装的系统中自带BBR加速的

BBR是 Google 在2016 年发布的一套拥塞控制算法。它尤其适合在存在一定丢包率的弱网环境下使用，在这类环境下，BBR 的性能远超 CUBIC 等传统的拥塞控制算法。

目前主流加速方式有2种：BBR和锐速。加速效果相近，对于海外的机器我是建议使用加速算法的，流量损耗大概在5%，但是效果提升巨大，目前已经成为大多数服务器自带的一个服务，自带的内核版本是Linux 4.9及以上的系统已经内置BBR但默认为关闭状态。目前CentOS 8.0+, Debian 9.5+都是这个内核以上版本，可以使用以下命令打开：

echo "net.core.default_qdisc=fq" >> /etc/sysctl.conf

echo "net.ipv4.tcp_congestion_control=bbr" >> /etc/sysctl.conf

sysctl -p

如果还嫌弃不够可以再打开TCP Fast Open：

echo "net.ipv4.tcp_fastopen=3" >> /etc/sysctl.conf && sysctl -p

所以大多数Linux系统都是我们的选择。搬瓦工提供的安装系统：

除了centos，debain，Ubuntu，还有一个AlmaLinux和rocky。这边不是很推荐最后两个系统，说是centos系统的后续接班人，但是相对来说文档不多，而且太小众，出现问题都找不到解决方案。然后在centos，debain，Ubuntu之间，我是会比较推荐debain系统，因为centos8系统生命周期2021年已经结束。虽然centos7还存续着，到2024年，但是系统毕竟属于后期维护，团队主力在centos stream。而且debain系统和centos系统相似（其实大多数Linux系统都是相似的，差别在一些包管理。最重要的是debain系统相较于centos系统，占用更小，系统更加稳定，128M的系统都可以完美运行debain系统。debain系统主打稳定性，建议一旦完美运行之后就别去再升级。文档来说相对于centos少，但是还是可以找到大多数的问题解决方案。其次如果是网站服务器，很多时候我们会使用控制面板，由它们来输入和执行命令，所以底层系统的差距对我们影响不大。

总结：如果你是新手（不折腾），使用centos，如果你有一定的排查问题能力且是追求性能极致的（生命不止，折腾不休），那么建议你使用debain。

2，控制面板选择

控制面板相当于我们和操作系统之间的一个中间媒介，用来控制服务器的文件读写，权限配置，php等其他程序的安装，让我们没必要修改一个文件需要打一大堆的命令行。面板能够提供：

  • 1，直观的文件操作和文档

  • 2，傻瓜化网站部署，反向代理和ssl证书申请等

  • 3，php状态查询还有一些插件的安装和卸载。

主流的控制面板程序有很多，但是都大差不离。这边主要推荐两个：AMH，宝塔。

宝塔：大概占用200M左右，国内版本需要绑定手机号，虽说有办法可以绕过去，但总归不方便。每次升级还需要操作一次，这边推荐国际版，内容一样，但是是英文的。可以满足大多数需求：提供一键安装望着那个环境，支持nignx的各种操作，提供软件源，可以让你安装其他一些适配的程序，安装好之后可以直接界面操作，而无需输入命令行。

AMH：之前是特定版本免费，现在已经是直接有免费版本了，根据官网的演示版本，他是更偏向于自定义的，很多东西不会默认给你安装，你需要一点点的去软件商店安装你需要的软件。自定义化程度高，但是相对的很多东西需要有一定的基础，软件商店提供的产品很多，很多都是知名的软件。上手成本高，但是最后成效是要好于宝塔，毕竟宝塔是自己开发，很多功能需要后面增加。这个直接采用主流软件挂载，基本可以实现所有操作。

推荐这两款主要是中文+有自己的文档社区。毕竟出问题好解决。

总结：如果你是新手请直接选择宝塔。该有的都有，该配的都给你配好了，即开即用

如果你是强迫症或者享受折腾的过程，那么请用AMH。

3，服务器程序

服务器内存和核心都是短板，所以一般我不会在服务器运行一些大型程序，基本上关于数据库的操作我都不会部署。目前几个程序几乎都是采用sqllite进行数据操作。人们购买服务器的想法多样：有些是想要部署自己的博客，有些仅仅是为了看YouTube，还有就是自己学习需要一台24小时不间断的电脑，侧重稳定性。但是我想说的是服务器能做到的不仅仅如此：基本上只要是采用python的服务，我们都可以部署，同样基于PHP的我们也可以做到。如果你需要，你可以在服务器上部署jupyter来进行远端程序开发，或者部署一个爬虫抓取对应的资料。这边举例几个：

  • jupyter：python开发工具，可以满足数据学习和爬虫开发，网页端运行。

  • 爬虫：基于python的自动爬虫工具，数据可以直接保存成json。

  • frp：一款反向代理工具，可以将你内网的服务发布到外网，让别人可以通过域名或者公网IP进行访问。

  • 博客：typecho或者其他基于python开发的博客。

当然，你如果喜欢也可以在上面安装docker，然后利用docker部署全部程序，实现完全独立的操作。

最后还是要提示一句，服务器稳定为主，不建议经常折腾，如若不是有必要，还是别换来换去的，还有，千万要记得备份，备份，备份。

结局：在一段时间后，我选择了重装系统，将系统换到Debain，面板采用宝塔国际版，英文界面，但是不影响使用，服务程序的重装比想象顺利，其中V2ray也进行了更新配置。

私人图书馆搭建-clibre

发表于 2022-05-01 本文字数： 825 阅读时长 ≈ 3 分钟

clibre是一个在线书籍管理工具，可以在线阅读PDF，EPUB等格式的图书，配合z-library，可以搭建一个属于自己的在线学习，阅读中心，而且还能管理每本书的状态，很适合在自己的服务器上部署一个。

clibre有两种搭建方式，python部署和docker搭建，根据个人情况选择。

python部署我使用的是官方搭建教程：https://github.com/janeczku/calibre-web，简单方便，然后配合宝塔，反向代理访问。但是官方的是没有对应的文档转换，比如pdf转mobi等功能，需要自己安装转换器。

docker部署的话，推荐使用https://github.com/lyy289065406/calibre-docker的docker，docker内部自带calibre的本地管理工具，还带官方的calibre-web的所有功能，作者还内建了转换工具。具体可以看作者写的部署文档，很清楚。

这边着重讲解使用宝塔的python管理工具搭建官方calire-web。

首先，下载对应的官方包

上传到服务器端，解压，记下文档路径。

下一步，在宝塔面板的软件商店里面查找python项目管理器，官方出品，但是只支持centos7.x。可以方便快捷的管理python版本。

安装完成进入程序之后，选择python版本，官方要求3.5+，这边我选择3.8。

路径选择上传到服务器后解压的路径，名称随意，框架选择python，启动方式选择python，启动的文件选择对应目录下cps.py

端口随意，记住就行，剩下勾选起来，安装模块+开机启动。

点击确定之后，系统就会在后台运行，你可以直接用域名+端口，或者IP+端口的形式访问。

初始用户名：admin 密码：admin123.

如果不想要后面的端口，可以使用域名的反向代理。正常创建域名之后，然后将域名反向代理到你的端口，这样就实现了完全的域名访问。

具体效果可以见https://book.minsummer.com/

使用下来，还是有点小问题：

1，是epub的文件如果太大，打开会太慢，然后每次放大缩小都会卡一会，因为屏幕大小变了，系统需要重新排版。这样导致有时候书签保存的位置也会乱。

PS:如果python项目管理器启动失败，有可能是因为系统安装模块错误，可以手动进入虚拟系统安装：

source 项目路径/项目名_venv/bin/activate

pip install requirements.txt

确保requirements.txt在你执行命令的目录下，否则你就要搬运过来。

python管理器默认会在你选择文件夹下创建一个虚拟环境，然后再在这里运行你的程序，对python虚拟环境感兴趣的话，可以看https://minsummer.com/IT/63.html这篇。

v2ray的一些Tips

发表于 2022-04-30 本文字数： 1.8k 阅读时长 ≈ 7 分钟

1，使用脚本安装V2，这个脚本内容放在GIT上，理论上是不会消失的，脚本里面覆盖了大多数的常用配置

bash <(curl -s -L https://git.io/v2ray.sh)

2，配置推荐Websocket+tls，这是目前可以使你的日常博客网站和v2共存的方案，域名可以推荐用子域名（tls使用泛域名解析），或者直接用主域名的特定路径。这个配置并不是简单的模拟，是完全正式的Https链接，所以和实际浏览网站没有任何区别，不会被认为是翻墙，所以一些人喜欢拿一个正常网站当掩盖，特定路径做V2代理。只要网站正常，一般不会有被封的风险。

3，Websocket+tls端口可以随便选择，也没必要开放，直接在宝塔面板里面使用对应的反向代理，如果是子域名直接反向代理写上http://127.0.0.1:特定端口。如果是博客特定路径，则需要填写上路径，然后对应的配置文件下：

“streamSettings”: {   “network”: “ws”,   “wsSettings”: {   “path”:

“/ray”————这里为你需要填写的特定路径，客户端上也要写上

根据：https://www.bookset.io/read/v2ray-guide/c6fb1ced8cb4e5d5.md 中的Nginx配置，我们需要增加两条在宝塔的反向代理中：

4，脚本安装的目录路径：

V2Ray 配置文件路径：/etc/v2ray/config.json——服务端配置文件，上面需要修改的就是这个地方 Caddy

配置文件路径：/etc/caddy/Caddyfile——宝塔面板一般使用nginx，不用这个可以忽略 脚本配置文件路径:

/etc/v2ray/233blog_v2ray_backup.conf——不用看，没有用

这边再增加一个：/bin/v2ray——这是v2的主要运行文件，其中有个：config.json，是用来配置一些默认参数的，用来优化V2，比如：bufferSize这个参数：每个连接的内部缓存大小。单位为 kB。用来均衡，服务器和你本地的网络之间速度不一样，简单说就是服务器速度快，提前下载好东西，然后再慢慢的传给你本地，内存越大，速度越快（提升有限），至少不用多次访问。还有其他参数可以看官方网站的本地策略，可以自己调整。

5，CDN可以隐藏真实的IP地址，通过这个策略来实现恢复锁住的IP，有免费的Cloudfare，但是访问速度看你的运营商。但是平时不打开，等到需要的时候，将DNS的状态改为代理，可以应急，访问自己被封住的IP。需注意，使用自动代理，1是有上传文件100M的限制，2是不是所有的端口他都会代理，如果你开放其他特殊端口，需要服务，请将这些端口:443/2053/2083/2087/2096/8443/80/8080/8880/2052/2082/2086/2095给那些你想要访问的服务，访问时就是http://域名：以上特定端口，就可以打开对应的服务，一般拿来做宝塔面板的后台。这样哪怕你IP被封了，也能正常操作。

6，宝塔面板是好东西，但是占用也大，推荐用，新版本需要绑定手机，但是可以下老版本，Github上有老的压缩包，毕竟这东西开源。

7，mckp协议可以解决一些运营商的掉包问题（比如移动），原理是多发，每次请求我发2遍甚至3遍回复，总有一条回复到的了你，缺点就是流量消耗大，是原来的2倍（理论值，算法优化下应该到不了），但是一样，提升有限，运营商的问题软件就算再努力，也是杯水车薪。

梅洛文集-<心理学与人类困境>总的来说并不是一篇严肃的科学论文,没有过多的实验数据,也没有很严谨的逻辑推论.更多的是作者在心理治疗上的一些思考:包括何为心理健康,何为自由,人类的差异性,已经心理治疗手段的局限性.正如标题所描述的,心理到底能多大程度的解决人类的困境,甚至于心理学这个学科的意义何在.

梅洛算是开辟了一个新的领域,存在心理学.他不再是简单的将人分为一类或者一个集合,他从根本是就确定,人是独立的个体,每一个都是特殊的存在,他们的生活,习性都是不一样的,我们无法像研究一个种群那样研究人类,甚至我们都无法找到我们的研究对象,这正是标题提出的问题所在,作为一个泛学科却没有一个可以长久有效甚至适用于大多数人的规则或者理念.作者试图在定位这一规则或者理念.他从一开始就确定了,人是特殊的存在,他是可以将自己分为主体,客体等多个身体,但是本质上他就只有一个个体,那就是人,人类的二分理念将人类限制在是否,有无,自我本我,而忘却了他首先是一个个体.在佛洛依德的影响下,大多数心理学家开始将人分成一个个小小的部分,但是往往在拼凑成一个完整体的时候发现这里少了一块,那里并不严实,作为一个新的想法,作者提出了存在理念,不再将人分成一个个小小的部分,也不再像传统心理学分析一样,探究他的童年创伤,认为人现在的一切都是源自于过去,所以过去会影响众大,甚至所有的毛病问题都是童年阴影的影响.

我并不是全盘否定这种理念,但是没必要将其夸大到所有的事情习惯都是童年的后续延展,也不必强求所有的问题都是过去的某件事导致的,个人觉得童年确实影响了每个人人生的大部分时间,但不是全部,在心智未成熟时期,确实过去发生的一切事情都会给个人的判断产生一定的影响,但是一旦个体有了自我的意识,或者主观想法,很多时候仅仅是为了生存于当下所作的改变.这会导致一部分异于常人思维的状况.作者着重强调了,区别神经症和精神性疾病的差别.某些神经症仅仅是个体为了保持自己的存在将自己置入其中,如果外部环境变化,那么个体的行为也会随之变化,以个体是为了存在作为主要的推论论据.

但是如果是为了存在所发生的所有事情都是正常的,合规的.那么这个社会就会乱套,所以又提出了一个何为心理健康的概念.健康对于人类来说并不陌生,但是如果有人要求给出一个定义,或者说一个现象,却并没有一个符合大众的想法.毕竟每个人的健康状态都是不尽相同的.所以,当个体陷入神经症时候我们并没有一个标准判断.作者也没法给出一个正确的判断标志.

作者同时也提出了现在心理治疗上的一些情况,很多时候病人和医生属于一种不平等的情况,甚至会发生病人迎合医生的病情,归其根本的是病人对医生的过分崇拜,很多病人对自己本身并没有一个充分的了解,尤其在心理上,一些症状或者一些情况,如果医生不加以诱导就会沉默.同样的也存在医生将自己置于上帝的角色,毕竟,对于心理治疗来说,治疗目标并没有一个明确的对象,所以往往需要医生自己主观去确定,但是个体之间环境不同,目标设立的也不尽相同.

综合来看,这本书属于晦涩难懂,毕竟不是一本正经的科普书籍,更多的是大师对存在心理学的一些想法和理念,总结归纳下是有一些东西,但是主管性质太强,不适合新手.

故事到了最后，小爆的思绪回来了。时间一分一秒的过去，但他的心情依旧没有什么好转，反而更加低沉了。看似不大的房子现在却如同平原一般，似乎刮着远处过来的风。小爆想入睡，但是睡不着，他昨晚梦见了那个笑容，梦见了他们在一起，那个梦真实的让他分不清是梦还是真的，那梦完美的让他怀疑这个世界的真实。然而梦终究会醒来。现在的他正处于梦的甜蜜和现实的残酷之中。说来也奇怪，以前的梦，虽然醒来之后就会忘记，但是这个梦即使过了这么久还是那么的清楚，清楚的好像如同昨天发生过的一样。

在经过了一番的胡思乱想，小爆的睡意又来临了。周围很安静，雨已经停了，偶尔传来远处的狗叫，或者是车子的咆哮。小爆想要入睡，他的梦让他更喜欢那个世界。

曾经，我追求长生不死，然而如今的我开始迷茫，正如他们说的，生活是一直都这么艰难，还是只是年轻的时候。里昂回答：一直是。我曾认为只要有网络，有一切，即使是长生也不会无聊。然而，在这个悠闲的早上，我却一点也不想坐，甚至有点怀念那匆匆碌碌的上班生活。即使自己忙碌一天，最后发现自己其实也没干什么。我似乎陷入了一个奇怪的循环。忙碌想要休息，休息觉得无聊想要忙碌。我似乎还没发现自己想要什么。或者说，在自己的梦想和生活之间没哟找到一个平衡点。我不是富二代，我只是一个苟活在这个世界的蝼蚁。我想要改变，但是现实太难，自己太懒，于是乎陷入在这浑浑噩噩的循环之中。我知道病因，但却治不了。无从下手。然而，就算是不去医治，未来的路也很长。忽然明白，比起明天就死去的病，为何人们更害怕那些煎熬许久的病。

我并不想探讨人生的真谛，也不想去责问什么，我只想要一个美梦，那梦中有我想要的笑容。

中午过后，小爆起床了，梦中发生什么，甚至有没有梦，我不得而知。

在这可怜的小单间内，小爆开始了他今天第一次的活动——起床。这是一间居民楼下的民房，坐落在这城中村中，随着人越来越往这繁华的都市挤，不少人选择住在这种价格便宜的单间民房。而有人的地方就有生意，于是乎，有人开始专门做这种生意，找个城中村老宅，加盖几层楼，装修下，隔层每间单间加个卫生厅，专门租给像小爆这样刚刚进入城市的大学生。由于这边价格低廉，交通还算是方便，所以刚毕业的小爆选择在这住下。

虽说是一间房间，但是由于在城中村，管理不严，加上周围都是加盖的楼房，所以导致这种房子采光几乎没有，通风也是很差。所以居住环境确实不算的上是好，夏天大太阳时是个闷炉，下雨是是个水壶，秋冬更不用说，冷的得盖上两床棉被，如果遇上少见的雨，恭喜你，你的风湿该犯了。

然而，即使这样，小爆也没有心思换房子，他们只不过是这个城市的过客，他自己心里清楚。这个城市提供给他工作，他在这里租个临时的住所。至于未来，他没有想过，或者说他不敢想，这个城市没有给他，他内心想着赚几年钱，然后回家。但他明白，这个只是一个痴人说梦而已。

迷茫的一代其实在每个年代都会出现，无论是90后，00后，甚至10后，只不过他们中的其他突出掩盖掉这个特点。现在的小爆就是迷茫的一代，拿着3000的工资，住在这600的房间内，加上水电，吃住，每月近2000的开销在这上面，还好，他父母并没有叫他寄钱回家，因为剩下的1000也会因为偶尔几次的放纵或者应酬而消耗掉。曾经有一年，小爆没攒下一分钱，最后靠着年终的一个月工资奖金才能得以回家过年。

小爆的情况并不是特例，事实上，他的同事，他的朋友也有不少这样的人，有的人更甚。他们是支撑着这个城市最底层——白领农民。如同农民一样，他们的工作一般是最底层的，或者说少有技术含量的，但是需求量巨大，比如文员。就是因为需求量巨大，所以他们比较好找到工作，而工资自然也不会高，至于未来的职业规划——并不是所有公司都是有上升渠道的，更多的是，这个工作你不干有大把的人来干，即使换个更高薪水的人来，也不会给你加薪的。暂且不说这些用人者的逻辑。反正小爆就是其中的一员。

他在14年毕业入职到这个公司，文员，普通而又大众的职位，工作的内容时而轻松，时而忙碌，有时候是录录单据，有时候又是在那和同事唠嗑打诨。这样舒适的环境一度让小爆觉得，工作不过如此，每天上班聊天还可以有钱拿。所以小爆的内心也就没有了跳槽的想法。他就这样，干着一份能拿钱的工作，然后过着一天又一天。偶尔的同事小聚，或者朋友的一起小资，会让觉得生活真是美好，虽然他大多数时候是在路边的小店吃着10块钱的套餐。

在麻利的收拾完自己之后，张小爆，拿起了手机，开始看微博，微信等等一系列的社交工具。如果评选21世纪最伟大的几项发明社交工具绝对是可以挤进前10。人们借助他向别人展示自己多姿多彩的生活，或者窥视别人的生活。这让我想起来我小时候总是会趴在窗前，看着邻居家窗户，因为我能在那边看到邻居家今天煮了什么，现在在干啥。而社交工具的出现，就是将我的窗户放进了手机，不一会就会有人将他们家的窗户传出来，极大的满足了我的好奇心。但是对于小爆，这个是他用来给别人展示自己的工具的，昨天的小聚，发个朋友圈，接受众人的赞和羡慕。来，看看，今天又有谁出去玩了，小爆聚精会神的看着手机，偶尔停下来，点个赞，或者评论几句。就这样时间一点一点的过去，人们在专注于某些事情的时候，总是会忘乎所以，包括时间。在长长的时间过去后，小爆终于放下了自己手中的社交工具。他的内心感到充实，因为他刚通过这个小小的屏幕和别人联系在了一起，即使他还在这。他明天可以跟同事聊一大堆今天在这个小屏幕看到的事情：你昨天去那里了，怎么样，好玩吗？这些总是能在周一的早上充满整个办公室。虽然他们一周七天有五天会在一起，但是还是会有两天的时间不能知道对方的消息，而这个社交工具满足了他们，让他们一直在一起。

在于同事们在精神上进行神交之后，时间已经来到下午一点，张小爆的肚子开始进行抗议。张小爆的小房间是有厨房的，额，或者更多的算是一个小平台，往往跟厕所连在一起，隔出一个小隔间，装上门，于是乎一个小小的单间就被分成卧室和厨房+厕所，还能进行干湿油烟分离，不能不说专门面向单身青年的住房，确实是有一定的设计思路的。但是由于条件和厨房限制，或者更多的是懒吧，小爆并没有在家煮饭。他选择了叫外卖。

外卖，这个其实并没有什么特殊之处，早在以前就可以将店里的东西打包带回家吃，不同的是，现在的外卖已经将其发展到极致，无需麻烦的带着工具上门打包，他们会使用一次性包装将食物送到你家门口，而你只需要等待即可。至少这极大的改变了小爆的生活，公司并没有食堂，所以每次吃午饭都是去楼下的店，由于公司位置偏远，所以店不多，但是公司人有很多，所以往往就得排起长龙，于是乎，他们选择了外卖，在11点点好之后，一般会在12点到达公司楼下，省去了排队的功夫。

小爆打开外卖软件，今天吃啥呢，饭？面？在思量许久之后，考虑了综合因素之后，小爆选择了平时吃得那家的面。还好，面对这个日常问题：今天吃什么呢？小爆没有考虑太久，因为太饿了。在干完这一切之后。小爆又陷入了无限的等待。等待着食物的到来。很多时候，我们本可以自己花费一点点时间解决掉某些事，但是我们却更喜欢选择去将事情交给别人，然后等待，即使此时的我们无事可做。也许他们看来耗费自己的体力比耗费自己的时间更可怕。

快餐到了，吃饭，吃饭的时候小爆总是喜欢看点东西，一些视频，或者是一些图片。不知道为何这样吃下去的饭总会比较香，也许是因为没人一起吃饭的孤独吧。但是今天他并没有什么想看的，他的脑子还在想着一些事，一些人。这让他的心绪有些混乱。就这样，他混混噩噩的吃完了他的午餐，虽然吃到最后他都没尝出味道来，也许现在的食物对于他来说也变得仅仅是食物而已，而不是辛苦劳作的一顿慰籍。

午后两点，雨早已停了，夏蝉也开始出来上班，知了知了的叫着。在夏天太阳的照射下，加上早上的雨，空气中有点湿气，温度有点高但却还算凉快，这算是夏天最舒适的天气了。但是在这叠积木一般的民房建筑群内，这种天气确实噩梦。早上的下雨，导致房间内满是潮气，但是又没有太阳照射将它挥发掉，也没有风将其吹走，就这样，留在这不足30平米的小单间中，墙上的瓷砖因为空气变得亮晶晶的，地面的瓷砖变得湿滑滑的。人在其中，往往会觉得黏糊糊的，但是又说不清楚是哪里。这样使得小爆十分暴躁。这也许就是无明业获得来源吧。但是现在这时间正是太阳最大的时候，出去那就是烤鸡。电风扇也不想开，温度还好，因为毕竟被旁边的房子挡住了阳光，没有多热，而且开起来吹过来的湿空气，让人更加烦躁。灯得一直开着，事实上，从早上他起来之后这个灯就没关过，这灯一到周末时间就是全天开的节奏。张小爆就在这样的房间里闷着，烦躁的他并没有心情静下来看个电影啥的，这是他以往打发时间的方式，也不想去床上睡觉，事实上，因为这湿潮的空气，床单和被褥也变得有点潮湿，让人的心情更加沮丧。尤其这民房没有顶楼阳台，被子啥的只能去民房建筑群外的广场，那里靠近马路，车来车往，但是也正是因为此，才能有阳光透过来。而平日里的衣服只能挂在唯一的窗台上，房东机制的用铝管做出来一个突出去的框架，形成一个突出去的防盗窗，既保证了安全，也给晒衣服一个空间。这种设计精巧，一物两用的地方在这间小屋子里可是有不少地方。

午后两点半，时间这就样一分一秒的过去，张小爆犹如一个被关在牢房的犯人一样，左右度步。虽然门外就是自由，但是那只有不知通向何方，他开始怀念公司了，当然怀念的不是工作，而是那已经被安排的一切，9点上班，聊天，娱乐，工作到12点，和同事吃饭，聊天，娱乐，午休，1点上班，聊天，娱乐，工作，到了晚上6点，下班，还能约上几个同事一起吃晚饭再回来。偶尔的部门聚餐更是锦上添花。然而这突如其来的空闲，却带来了无尽的寂寞，他排解不掉。事实上，他一直让自己如此的忙碌，而没有时间静下来想想事情。但是正好今日赶上这天气，这时间，他无所事事，但又没有地方浪费他那宝贵的时间，他开始发呆，思绪曾经飞向雨滴，他思索过去，但是却沉溺其中无法自拔，他知道该放下过去，但是记忆没有放下，自己没有放下。他又开始考虑未来，发现那里荒芜一片。他在过去未来现在不断的穿插，他感觉自己即将发疯，马上就要失去了理智。于是乎，他决定了，和别人一起出去玩，他翻开他的通讯录，名单很长，但是可以在这个时间打的人很少，大多数客户不可能打，同事，不行，他们也有自己的周末，而且谁也不想在一周五天之后在第六天又见到。朋友，并不是没有，聊得来的少，大多数都是出于礼貌加的。在搜索了一圈之后，他发现没有人可以约，也就是说他只能接着在这个小屋子内直到自己发狂。他看了下时间，三点10分，看来也是有好事的，时间在刚才的臆想中过去了不少，等到四点，等到四点，等到四点就出去吃晚饭吧，顺便转转。这也许是接下来最值得期待的事情了，既然那有力目标，接下来的时间就变的无所谓，小爆打开他的电脑，这台电脑从大学跟着他来到了这间出租屋。随着时间的流逝很多东西都变了，唯一不变的依旧是这台电脑依旧在他身边，依旧是原来的系统，只是有点卡顿，不过在处理小爆的闲暇时间是足够了。

时间过得飞快，三点40，张小爆开始准备，因为他早就不想在这超时闷热的房间呆着，他想要出去，享受那个花花的世界，他花费十几分钟时间，穿戴，打扮，然后出门。

民房建筑群的路，因为周围房子的占道，早就变得扭扭曲曲，加上人多，不注意保护，路上早已坑坑洼洼，虽然经过中午的太阳，但是依旧能在路面的坑洼出见到早上的雨水，而且由于路面狭窄，阳光照射时间短，地面还是有点潮湿。从小爆住处到达最近的商场需要五分钟，在走过一条条小路之后，转过一个角，豁然间，出现一条柏油马路，高楼耸立，绿树阴阴，干净的街道，整齐的路灯，甚至不远处就是一个标准的高端写字楼，这就是城市，真正的城市。与小爆身后的形成鲜明的对比。在小爆还没来到民房之前，他对城市的见解，更多的是，高楼，大厦，柏油马路，单身公寓。而在真正的找到这个住处之后，他才明白，那些，都有，只不过不是他的。他第一次见到原来如此美丽的城市里面有着如此肮脏不堪的地方，犹如皮藓一般，在这个美丽的都市中显得如此的格格不入，一开始他是拒绝的，甚至觉得这里是一些闲杂无业游民的聚居地。然而命运却与他开了个玩笑，最后他看中了这里低廉的房租，住了进来。电视上光鲜亮丽的高楼与他无关，他的公司是附近工业园内的厂房，嘈杂且充满灰尘。白领干净整洁的制服也与他无缘，他们公司根本就不发制服，最后，那舒适的单身公寓也是遥不可及，最便宜的一个月经1500的房租，对于3000+工资的他根本承担不起。最后，在同事的介绍下，他找到了这边的民房，便宜，齐全。

进来居住之后，小爆才发现，这里的住户其实并不是都是些社会闲杂人员，相反，更多的是和他一样的白领，从每天早上去上班就能发现，一个个穿的光鲜亮丽，跟这里的环境格格不入，更有甚者穿着笔挺的西装制服，匆忙的上公交去。至此，张小爆就不在排挤这个地方，他甚至设身处地的为这些人想：也许他们和我一样，工资承担不起，只能居住在这。

去商场的路上，能遇见很多人，他们有些很眼熟，好像是早上上班的照面，有些很开心，一看就是刚刚happy回来，也有一些和小爆一样，急不可耐的想出去拥抱那个花花世界。路很短，人很多，他们或并行，或前后脚，总之，能感觉到对方和我的目的是一样的。虽然已经是下午四点了，但是太阳依旧很热。

1，使用脚本安装V2，这个脚本内容放在GIT上，理论上是不会消失的，脚本里面覆盖了大多数的常用配置

2，配置推荐Websocket+tls，这是目前可以使你的日常博客网站和v2共存的方案，域名可以推荐用子域名（tls使用泛域名解析），或者直接用主域名的特定路径。这个配置并不是简单的模拟，是完全正式的Https链接，所以和实际浏览网站没有任何区别，不会被认为是翻墙，所以一些人喜欢拿一个正常网站当掩盖，特定路径做V2代理。只要网站正常，一般不会有被封的风险。

3，Websocket+tls端口可以随便选择，也没必要开放，直接在宝塔面板里面使用对应的反向代理，如果是子域名直接反向代理写上http://127.0.0.1:特定端口。如果是博客特定路径，则需要填写上路径，然后对应的配置文件下：

6，宝塔面板是好东西，但是占用也大，推荐用，新版本需要绑定手机，但是可以下老版本，Github上有老的压缩包，毕竟这东西开源。

7，mckp协议可以解决一些运营商的掉包问题（比如移动），原理是多发，每次请求我发2遍甚至3遍回复，总有一条回复到的了你，缺点就是流量消耗大，是原来的2倍（理论值，算法优化下应该到不了），但是一样，提升有限，运营商的问题软件就算再努力，也是杯水车薪。

8，服务器建议买cn线路，线路是重中之重。好的线路可以极大的加快你的速度，现在流行的是香港和日本这些靠近的国家，但是香港的线路染上 国内的恶习，要么带宽限制1m，5m啥的，要么就是流量100g收费，这边建议使用搬瓦工的CN，线路，运营商历史悠久，而且速度还不错。主要是带宽1G，流量1000G/月起，带宽主要是服务器和你之间的最大速度，由于服务器一般线路优化，访问大多数网站速度都很快，曾经将谷歌硬盘内容下载下来，达到满载，被系统以为是受到攻击了，流量就是你用这个服务器上传和下载的流量，一般来说一个人100g /月算是很充足。配置建议1G内存以上，512M是便宜，但是无法安装Docker容器管理器，这是个神器，方便简单的管理你的docker，现在很多服务都是直接一个docker解决，方便管理，而且搭建一些网站也需要一定的负载能力。至于最后的硬盘容量，没必要太较真，因为你不可能在上面存放大量的资料，10g就足够大部分的需求，主要是一些软件，至于想在上面放电影的同志，不是很建议，主要因为电影有可能会触发盗版提示（毕竟你服务器在美国）。很多时候我们哪来当中转站，一些文件可以先下载到服务器，然后再从服务器下载到本地。这速度和你直接拿V2做代理是不一样的。

9，wordpress是个好东西，但是如果你是拿来玩玩，做个简单博客，可以搭建在服务器上，其他用途，请找一个wordpress的托管商。wordpress相当于是一个框架，里面有几千款的插件可以选择，所以对系统要求很大，而且她要求有对应的数据库，其中Mysql数据库要求1G内存，所以很多时候这个东西会占用你很多的资源，个人轻量博客可以，复杂的电商最好用专业的托管网站更合适，至少不会卡。

10，有时候配置好后，无法访问，请按照：客户端——cloudfare——服务端  顺序排查问题，客户端配置有没有问题？请去cloudfare看下是不是打开了underattack模式，这个模式下，会把所有的链接都拒绝一下，防火墙可以看到拒绝的日志，看下是不是被系统拒绝了，最后再去确认下IP是不是被封：随便找个ping IP的网站，国内ping你的IP能不能访问就可以。确定被封了，打开cloudfare提供的cdn，继续happy。

11，windows客户端v2使用速度测试的时候会发生服务端无法访问，直接打开网页访问谷歌最合适。

12，有时候访问不了，仅仅是证书验证的问题，可以试试在客户端里打开跳过证书验证，或者放空。

vscode搭建虚拟环境

发表于 2022-04-30 本文字数： 462 阅读时长 ≈ 2 分钟

1，python下载之后第一件事就是更换国内源。

pip config set global.index-url  https://pypi.tuna.tsinghua.edu.cn/simple

2，python3.8之后系统自带虚拟环境，直接调用即可；

python -m venv 路径/文件夹

3，vscode安装好之后，第一步在拓展里面搜索Chinese，安装之后即可是中文，然后是python插件

4，虚拟环境创建之后，打开对应的文件夹，里面有Script的文件夹，这里就是我们虚拟器的专用python，进入虚拟环境使用

#如果使用的终端是 cmd 命令提示符，需要运行「activate.bat」文件

虚拟环境\Scripts\activate.bat

#如果使用的终端是 Windows PowerShell，需要运行「activate.ps1」文件

虚拟环境\Scripts\activate.ps1

两者都可以通过右键复制路径，快速启动，进去之后再去安装对应需要的包

5，如果提示：在此系统上禁止运行脚本。使用

set-executionpolicy RemoteSigned

然后选择Y即可。

6，进去之后，在左下角的解释器里面选择对应虚拟环境下的python.exe即可，如果是jupyter notebook的话，需要提前安装ipython包和jupyter包，然后创建1.ipynb的文件，右键选择：重新打开编辑器的方式。即可选择jupyter notebook。

如果有时候没有出现，可以使用crtl+shift+p，输入select interpreter：选择解释器。选择虚拟环境下Scropts下的python。或者重启一下Vscode。让系统再次寻找到解释器即可。

7，vscode是好东西，机器学习也不难，实现不难，就是要知道它为何这么实现需要很多数学知识，而且由于对其原理不是很清楚，很多时候没办法将其应用在一些自己有需求的方面，所以如果不是以此为生，更多的是兴趣和了解

何为自由-心理学与人类困境

发表于 2022-04-30 本文字数： 1.2k 阅读时长 ≈ 4 分钟

梅洛文集-<心理学与人类困境>总的来说并不是一篇严肃的科学论文,没有过多的实验数据,也没有很严谨的逻辑推论.更多的是作者在心理治疗上的一些思考:包括何为心理健康,何为自由,人类的差异性,已经心理治疗手段的局限性.正如标题所描述的,心理到底能多大程度的解决人类的困境,甚至于心理学这个学科的意义何在.

梅洛算是开辟了一个新的领域,存在心理学.他不再是简单的将人分为一类或者一个集合,他从根本是就确定,人是独立的个体,每一个都是特殊的存在,他们的生活,习性都是不一样的,我们无法像研究一个种群那样研究人类,甚至我们都无法找到我们的研究对象,这正是标题提出的问题所在,作为一个泛学科却没有一个可以长久有效甚至适用于大多数人的规则或者理念.作者试图在定位这一规则或者理念.他从一开始就确定了,人是特殊的存在,他是可以将自己分为主体,客体等多个身体,但是本质上他就只有一个个体,那就是人,人类的二分理念将人类限制在是否,有无,自我本我,而忘却了他首先是一个个体.在佛洛依德的影响下,大多数心理学家开始将人分成一个个小小的部分,但是往往在拼凑成一个完整体的时候发现这里少了一块,那里并不严实,作为一个新的想法,作者提出了存在理念,不再将人分成一个个小小的部分,也不再像传统心理学分析一样,探究他的童年创伤,认为人现在的一切都是源自于过去,所以过去会影响众大,甚至所有的毛病问题都是童年阴影的影响.

我并不是全盘否定这种理念,但是没必要将其夸大到所有的事情习惯都是童年的后续延展,也不必强求所有的问题都是过去的某件事导致的,个人觉得童年确实影响了每个人人生的大部分时间,但不是全部,在心智未成熟时期,确实过去发生的一切事情都会给个人的判断产生一定的影响,但是一旦个体有了自我的意识,或者主观想法,很多时候仅仅是为了生存于当下所作的改变.这会导致一部分异于常人思维的状况.作者着重强调了,区别神经症和精神性疾病的差别.某些神经症仅仅是个体为了保持自己的存在将自己置入其中,如果外部环境变化,那么个体的行为也会随之变化,以个体是为了存在作为主要的推论论据.

但是如果是为了存在所发生的所有事情都是正常的,合规的.那么这个社会就会乱套,所以又提出了一个何为心理健康的概念.健康对于人类来说并不陌生,但是如果有人要求给出一个定义,或者说一个现象,却并没有一个符合大众的想法.毕竟每个人的健康状态都是不尽相同的.所以,当个体陷入神经症时候我们并没有一个标准判断.作者也没法给出一个正确的判断标志.

作者同时也提出了现在心理治疗上的一些情况,很多时候病人和医生属于一种不平等的情况,甚至会发生病人迎合医生的病情,归其根本的是病人对医生的过分崇拜,很多病人对自己本身并没有一个充分的了解,尤其在心理上,一些症状或者一些情况,如果医生不加以诱导就会沉默.同样的也存在医生将自己置于上帝的角色,毕竟,对于心理治疗来说,治疗目标并没有一个明确的对象,所以往往需要医生自己主观去确定,但是个体之间环境不同,目标设立的也不尽相同.

综合来看,这本书属于晦涩难懂,毕竟不是一本正经的科普书籍,更多的是大师对存在心理学的一些想法和理念,总结归纳下是有一些东西,但是主管性质太强,不适合新手.

开头，为我们描述了一个充满营销，广告，的异世界。他们强迫你看广告，并因此收取里程，而这本来就是我们应有的拒绝权利，让我不禁联想到真实的社会，广告商无孔不入，劫持路由，网页，公交，一切的一切都充满了广告，如今的广告已经差不多做到眼之所见，广告之所及，未来的发展路线也许真的如片中所 描述，也许还会检测你眼珠是否观看，然而这一切正如主角最后爆发所讲的，都是虚伪的，都是虚无，都是空的，没有一点点的真实感。唯一让他感到真实的是那份“爱”，但是，最后也被摧之殆尽，没有一点点的剩余。

当剧情走到了那唯一的真实感被毁灭之后，我一时也猜不到接下来的剧情，我甚至怀疑会以主角的自杀结束，甚至黑暗的我想着主角的死亡才会使得这一集得到升华，然而，我错了，编剧采用另一种方式为这一集得到了升华，只不过是黑暗的升华，我们的主角开始攒钱，努力的攒钱，为了第二张的门票，他吃剩饭，少用牙刷，减少不必要的花销，与一开始那个无所顾虑消音形成反差，在我看到他拿起那个玻璃的刀时，我以为他是准备搞点大事，比如杀了那三个主持人，为摧毁自己真实进行复仇。然而，他只是将刀指向了自己，是的，指向了自己，并且爆发了，一场看似淋漓的爆发，最后却也在三个主持人的唆使下，陷入了更加无尽的黑暗，讽刺的是最后那把本以为用来结束一切的玻璃刀，最后却成为了他吃饭的工具，看到他轻轻的将其放在盒子内，我觉得这是本集的另一种提升吧。看完后，我眼里所剩的也只有黑暗。

好多事情好悲伤

发表于 2022-04-30 本文字数： 394 阅读时长 ≈ 1 分钟

好多事情好悲伤，悲伤的我都无法写下来。

我希望将自己的点点滴滴写下来，写一本自己的史书，不是日记，是史书，一本自评功过的史书。

但是我发现好多事，好悲伤，悲伤的我无法写下来，我想，等到我平复，能正视这些事情后再写下来，然而，到最后，我更多的是想要忘记这些事情，好悲伤，悲伤的我想要忘记。

我无法描绘自己悲伤的画面，因为那时，我在其中。我唯一知道的是，那时候的我从未感受到人生如此的漫长。

好想有一种药

喝下，就会忘记一切

好的，坏的，想忘记的，想留下的

然后，我会喝下

末了，说句：真幸福

这就好像将自己完完全全的剥开，没有留下一点点和隐私，没有任何的借口和其他可以隐藏，有的只有对这件事情最真实的看法

好多事情真的好悲伤，悲伤的我好想结束自己，因为这样，我就不会存在，也没有存在过。

我往往追求的是一件事物最正确的答案，这样是对的，也是正确的，但是一千个正确的重复之后，我总是有种错误的想法，为何，也许仅仅是因为想吧，那突如其来的想法总是让人着迷而且迫不及待，比如此刻，想要细腻而且真挚的描绘自己的内心。

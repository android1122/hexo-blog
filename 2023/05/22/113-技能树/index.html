技能树梳理

发表于 2023-05-23 阅读次数： 本文字数： 420 阅读时长 ≈ 2 分钟

梳理一下自己的技能树

技能树

开发

  • 程序开发
    • python
      • 实现应用
        • 数据挖掘
          • 没有数据可以挖掘
        • 爬虫
          • 没有合适的需求
        • 简单的AI
          • 底层的ai作用不强且没有较好的运用，大规模ai需要数据庞大
      • 高阶
        • 学习路线图

运维

  • docker
    • 服务器运维和自建docker

数据分析

  • excle
    • 满足目前98%的需求
    • 已经搭建一个可以直观动态的数据表
  • power BI
    • 满足目前98%的需求
    • 已经搭建一个需要手动更新的多维度数据表
  • python-pandas
    • 满足目前98%的需求
    • 已经搭建一个需要手动更新的数据表

应用树

数据记录和显示

  1. excel
    • 优先使用，能解决大部分问题
  2. power BI
    • 处理大规模数据或者图形化
  3. python
    • 偏向服务端，无法直观显示数据，但是可以用来挖掘数据内在联系

数据处理-自动化

  1. python
    • 方便处理excel等
  2. excel
    • 宏/vba，也可以处理，内在处理方式更加安全

爬虫

  1. python
    • 日常应用
  2. excel
    • 未实践，不过可以用在在内建表内显示目前情况
  3. power BI
    • 未实践，用来显示参考数据

期望树

  • 但当涉猎，见往事尔：每样都涉及一点，都了解一点，不一定需要了解透彻
    • 前端开发：php
    • 数据挖掘书
  • 学不在广，在深在精：如果发现有东西可以应用落地，可以深入了解
  • 绝望之墙：没有应用项目
    • 破墙之路：兴趣爱好
    • 破墙之路：工作需求

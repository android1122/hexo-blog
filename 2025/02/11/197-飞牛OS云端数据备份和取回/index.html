飞牛OS云端数据备份和取回

发表于 2025-02-12 阅读次数： 本文字数： 1.4k 阅读时长 ≈ 5 分钟

我算是比较早一批用云盘的用户，更多的是用来备份上班时候的数据，方便加班或者电脑没带的时候简单处理下。私心就是离职的时候我就不用费劲巴拉的拷贝文件。这些年主流的几个网盘程序都使用过了，趁现在自己买了一个NAS，想要把数据都回笼，所以趁着这个机会也来评判下用过网盘的优劣：

1. 百度云

大名鼎鼎，我是早期用户，现在空间2T，我在里面存有毕业后1-4公司的资料，占用600GB。说重要么？也不是很重要，但是毕竟是自己当初一点点累积的，就想着下载回来。我一直用到百度网盘限速之后就不在使用，因为当初换到第5家公司，我发现下载下来花费的时间超级长，就暂且搁置了。

数据下载：
飞牛OS支持直接挂载百度云盘，所以使用很简单。下载可以使用备份计划直接拉回来。速度么？不敢恭维，我前后一个春节在备份，差不多20天，现在只下载了76GB的数据，初略估计还得几个月。

好处是上传没有限制速度，我将OneDrive的数据上传上去只花了几个小时，差不多36GB.

百度云盘的会员分为好几种：

最便宜的一个月40，我的想法是：后期将这2T的云盘当作容灾备份，万一出现极端情况，我可以花40块钱，将数据全部下载回来，不至于啥都没有。

2. 坚果云

我在坚果云里面存放第4个和第5个公司的文件，数据不多6GB左右。坚果云的优势是不限速，但是限制上传和下载的流量，上传1GB，下载3GB每个月，我预期取回所有数据需要2个月。

所幸数据不多，如果是百度云的600GB那么估计要1年多才可以取回。当初是百度云限速才转换成它的，毕竟工作数据一个月不会太多，完全是满足的。
坚果云的会员也是最奇葩的，30块钱一个月，但是空间30GB，不限速。聊胜于无吧。

这次花2个月把数据搬回之后，我应该是不会用它的。

3. Onedrive

我买的是家庭版，每年分担下来4块钱，1T空间，不限速，自带客户端好用，而且还可以回复历史版本。付费的东西就是好。目前使用数据37GB，存放的是我现在公司的数据。免费版本的是5GB，我是很推荐新入职场的用户使用这个东西。
飞牛OS目前没有像群晖那样有个cloudsync可以进行网盘互传，所以我使用的是OneDrive的Linux版本，文档挺详细的，但是好像并不是官方的，有点是个人爱好者。
搭建教程：
docker部署
docker官方提供了很多版本，我选择默认的latest，需要根据你的硬件进行选择。

你可以选择直接登录飞牛的SSH，然后复制粘贴以下命令即可。
>docker run -it –name onedrive -v onedrive_conf:/onedrive/conf
-v “备份路径:/root/onedrive”
-e “ONEDRIVE_UID=1000
-e “ONEDRIVE_GID=1000
driveone/onedrive:latest

或者你可以使用飞牛自带docker进行部署。需要注意，如果你使用SSH进行部署，系统会弹出一个链接，你复制链接按照要求登录即可，最后在粘贴返回的链接即可。如果你使用的是飞牛docker进行部署，你则需要进入终端，重新执行下同步命令：

onedrive -s
或者
docker exec -it 容器名称 onedrive -s

这里有几点需要注意：
OneDrive的默认下载路径是和你登录进去的用户有关系，默认是~/OneDrive，所以你的备份路径需要根据实际情况进行调整，

onedrive –display-config
查看当前运行的配置

这个容器可以直接调用比如上面的 docker exec -it 容器名称 onedrive -s，也可以做长期监控

onedrive -m

具体可以看软件官方文档
我这边的配置是将ROOT下的.CONFIG文件一起映射出去，保存在docker外，否则会出现，每次重启docker就会要求你重新登录。

总结

容灾备份算是玩NAS的必备条件，有些人喜欢使用UPS来保证数据安全，但是我还是选择直接使用云盘，它满足了，异地，异介质双重要求。除开那些几百GB的高清电影，其他东西我都会选择放在云端。最差的情况下，只需要40块钱开通一个月就可以拿回全部数据。

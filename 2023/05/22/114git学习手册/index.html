git简明指南

  1. git工作流
    1. 本地git add到暂存区
    2. 暂存区git commit到HEAD
    3. HEAD的内容git push后才会到你的远程仓库
  2. 创建/复制
    1. git init
    2. git clone
  3. 增加/删除文件
    1. git add
    2. 或者直接增加
    3. git rm
    4. 或者直接删除
  4. 回退版本
    1. git reset
      1. –soft 回退上一次修改的内容
      2. –hard 回退所有内容
      3. HEAD~0 【0-n】回退到之前第几个版本
  5. 分支管理
    1. git branch 创建分支
    2. git merge 合并分支
    3. git checout 切换分支

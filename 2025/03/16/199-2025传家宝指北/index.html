2025传家宝指北

发表于 2025-03-17 更新于 2025-01-04 阅读次数： 本文字数： 2.9k 阅读时长 ≈ 11 分钟

不知不觉传家宝系列已经来到了第三个年头，依旧还是和之前一样要求：

  1. 持有价格够低，年付200人民币以下算好，100以下算神机，年付300以下算好机子，年付200算是神机价格合适，一分钱一分货，低价的话好多野鸡厂商也能进来。
  2. 够稀有，一般都是下架了，买不到目前
  3. 线路好，毕竟富强是第一需求
  4. 配置合格，至少能翻墙没有明细的短板

这次按照厂家进行分类，时间进行排序。一个是方便知道哪些厂家今年出现了神机，一个也能看出来厂家的态度如何。

搬瓦工

瓦工自从被DMIT和V.PS冲击之后，今年连续两次发力，一个是DC9机房，一个就是

1月14日推出：

套餐名称 机房 配置 带宽 流量 价格 人民币 备注
POWERBOX DC99（USCA_99） 1核1.5G内存30G硬盘 2.5Gbps 1500G/月 折后41.95美元/年 303元 /年 只有DC99一个机房

神机，但是问题很明显，就是不提供任何帮助，自我管理，而且支持30天内IP被封退款。
3.19日，瓦工确定DC99有线路保证，且允许发布工单补差价迁移至DC1线路。DC99机房是三网GIA，DC1是GIA+CMIN2，有点类似DMIT家的Pro和EB系列，但是DC1单从线路上来看肯定是优于DC99的，但是每个人网络情况不同，可以多方位选择。

3月份推出

套餐名称 机房 配置 带宽 流量 价格 人民币 备注
NODESEEK-BIGGERBOX-PRO DC1 洛杉矶 1核1G内存20G硬盘 2.5Gbps 1000G/月 折后36.36美元/年 264元 /年 即使 IP 被 GFW 阻止，30 天内也可退款
NODESEEK-MEGABOX-PRO DC1 洛杉矶 2核2G内存40G硬盘 2.5Gbps 2000G/月 折后45.68美元/年 331元 /年 即使 IP 被 GFW 阻止，30 天内也可退款

线路：洛杉矶 电信CN2 GIA、移动CMIN2、联通GIA/CMIN2，去程：电信GIA、移动CMIN2（部分地区走AS1299）、联通4837直连(北京出口)

这款是论坛NODESEEK联名，确实出来的产品都是解决用户痛点的。要流量有流量，要口子有口子，要配置也有配置，满足你所有的需求。神机无疑，而且这个是有线路保证和工单的。价格降低的同时，线路好了。而且瓦工最大的有点就是口子给的大，都是2.5G的带宽。相较于V.PS家的，他们的优点就很明显。如果说唯一美中不足的就是：不支持DMIT的IP被墙15天内免费更换。
免费换IP就是一个承诺，哪怕你用不着，但是有和没有是两个心理。我可以不用，但是你不能没有。

5月份推出-7月份又重新推出

套餐名称 机房 配置 带宽 流量 价格 人民币 备注
MINICHICKEN DC1 洛杉矶 1核1G内存20G硬盘 1Gbps 1000G/月 折后17.71美元/年 127元 /年 即使 IP 被 GFW 阻止，30 天内也可退款

线路：普通线路，不建议购买，速度啥的没啥优势。
这个和之前的19刀的传家宝一样，不过之前内存要小一点512M，这个算是更新下配置重新上台，不过也进一步验证了，那些古董传家宝没有留下来的必要了，现在机器和线路指挥更价的便宜。这机器就是便宜，如果你有一台中转的，那么可以考虑，其他的，不是很建议，我之前用过几年19刀的洛杉矶线路，也是无优化，一言难尽，有得选肯定不买。

NODESEEK-MEGABOX-PRO 重出江湖

套餐名称 机房 配置 带宽 流量 价格 人民币 备注
NODESEEK-MEGABOX-PRO DC1 洛杉矶 2核2G内存40G硬盘 2.5Gbps 2000G/月 折后45.68美元/年 331元 /年 即使 IP 被 GFW 阻止，30 天内也可退款

可以入手，我直接入手一个，三网优化，还要啥自行车，流量，口子都可以。

6月份推出

套餐名称 机房 配置 带宽 流量 价格 人民币 备注
The Amsterdam Plan 荷兰阿姆斯特丹DC1 1核1G内存20G硬盘 2.5Gbps 1000G/月 折后36.36元/年 261元 /年 即使 IP 被 GFW 阻止，30 天内也可退款

线路：阿姆斯特丹 GIA+9929+CMIN2 （目前是3网GIA，联通和移动的等过白）
三网优化，目前是GIA，后续会更改，什么说，这个市场终于卷起来了，我是举双手赞成的。如果你仅仅是要翻墙，这个会比MEGABOX更加合适。可惜手里太多机器，这款就让了。这款差不多卖了3个小时就没货了。但是不需要太担心，按照瓦工的尿性，迟早会补货，这次不过是探探路。

总结：瓦工终于开窍了，在前两年搬瓦工是高端的象征，一台HK8的机房，90+美元。但是这两年V.PS和DMIT不断推出低于50美元价格的服务器，在价格面前，什么都是浮云。而且瓦工的机器很多时候是属于隔靴搔痒。要么配置太低，要么就是不保证。翻看前年的机器都没有什么值得推荐的，看去年的出了35刀的机器，但是内存不足1G。BOX系列又是不能保证。终于有了两款要线路有线路，要配置有配置。任君挑选。我个人是十分看好NODESEEK联名款。
3.19已经DC99宣布进行线路保证了。这无疑是个好消息，而且越来越对标DMIT家的两个系列。

V.PS

在搬瓦工和DMIT的打击下，这家厂商已经有点后继无力的感觉。尤其年初出现了香港机房断线的事情，虽然最后退款和补偿到位，但是还是没法解决用户感受。毕竟做不到你可以不做。现在香港的线路没有一家大厂有价格低于100美元的。
V.PS今年声音很小，就出了一款东京的高级线路，三网优化但是价格太夸张了，255欧元/年。

后续会持续更新，有合适得就继续更新！！！

悲报：3月21日，圣何塞的移动CMIN2拔线，改走GIA或9929，虽然影响不是很大，但是总归是从传说中的三网优化神机变成了两网。用还是能用，但是没有之前好用。

总结：V.PS在DMIT出现之前，算是少数的大厂，尤其是它是瓦工HK机房的上游，这让它的口碑不错。那几年顺势推出几款传家宝，比如45欧元3年。但是问题也很明显，官方自己下场炒鸡。很多时候一些好的价格贵得机器，都在黄牛手里。而且溢价离谱，都是1000，2000得溢价。所以我个人内心是对这家比较反感得。目前还有它家SJC得三网优化机器，带宽600M，流量500G。说够也够，但是万一某个月看得视频多了就不足了。
V.PS这家实力确实差了点，只能看它后面还有什么戏吧。

DMIT

大厂，标准得大厂。而且量大管饱，很多时候卖的机器能持续一两天，不存在黄牛全包得下的。机器量多而且品种多，任君选购，还有30天一次得被墙免费换IP服务，大气。作为对比，瓦工换一次IP是7刀。它家得机器是溢价最少得，因为它时不时得补货，而且保有量大。
它家今年也出来了几款机器，除开几款高端机器，很多都是补货，而且今年2月18号推出了流量超出不停机服务，根据配置和价格超过流量之后，速度最低可以保持2Mbps。

LAX.Pro.MALIBU 洛杉矶 1核1G内存20G硬盘 1Gbps 1000G/月 49.9美元/年 355元 /年 3网CN2GIA

总结：DMIT绝对是大厂，硬气且产品线丰富，而且它很与时俱进，我们可以拭目以待，在3月份瓦工刚刚推出得NODESEEK联名款之后，它肯定有应对。到时候我会更新这篇文章文章。

总结

翻看前两年得传家宝指南再看看今年。我们可以确定的是随着技术和线路得成熟；产品得价格已经慢慢下跌。V.PS已经卷不动被踢出主流行情，剩下得就是看今年DMIT和搬瓦工得对决。
截至目前位置，T1级别得机器就是这两台，更加推荐MEGABOX，加量，但是价格加的有限，十分合适。

套餐名称 机房 配置 带宽 流量 价格 人民币 备注
NODESEEK-BIGGERBOX-PRO DC1 洛杉矶 1核1G内存20G硬盘 2.5Gbps 1000G/月 折后36.36美元/年 264元 /年 即使 IP 被 GFW 阻止，30 天内也可退款
NODESEEK-MEGABOX-PRO DC1 洛杉矶 2核2G内存40G硬盘 2.5Gbps 2000G/月 折后45.68美元/年 331元 /年 即使 IP 被 GFW 阻止，30 天内也可退款

当然如果溢价太高也不需要强行上，毕竟传家宝年年都有，我会建议你转投DMIT得产品，量大溢价少，购买方便。没什么特殊需求买Pro系列。如果家里用的网络是移动或者移动接触比较多那么就用EB系列。两个都是很好的线路，差距不会很大。比如我用移动和联通得网络多，我就直接用EB系列。
最后给一个简单得传家宝判断依据，只要现在低于等于LAX.EB.WEE价格得，且配置差不多得，你都可以第一时间入手，如果你正好有需求得话。
